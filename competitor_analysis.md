## 😡 What Customers Are COMPLAINING About (1-3 Star Reviews)

### 🔴 #1 — Excessive Ads / Intrusive Advertising (30.8% — 3,865 reviews)
**THE #1 PAIN POINT BY FAR.** Nearly 1 in 3 negative reviewers complain about ads.
- Full-screen ads on every tap / file open
- Unskippable video ads blocking document access
- Ads disguised as buttons tricking users
- Ads playing even in paid/premium versions
- Pop-up browser ads, notification spam ads

> *"it is not an app it's an ad show. Useless"*
> *"Loading file too slow. Every new action you tap, ads. Really not suitable to open docs in a hurry."*

---

### 🔴 #2 — Editing Features Missing or Broken (5.0% — 634 reviews)
- Can't edit Word/Excel/PPT files — only read-only
- No annotation, highlighting, or markup tools
- No ability to fill forms or sign documents
- Users forced to use a second app for editing

> *"I cannot edit my old files and can't create new file. Good for reading and file management only."*

---

### 🔴 #3 — PDF Issues (4.7% — 594 reviews)
- PDFs fail to render properly
- Can't zoom on small text
- PDF merge/split/convert features broken
- Landscape/rotation issues with PDFs

> *"It couldn't read my pdf file unfortunately"*
> *"It won't even let me view a pdf anymore"*

---

### 🔴 #4 — Can't Open / View Files (4.4% — 555 reviews)
- Files fail to open with cryptic errors ("File Parsing Error")
- Specific formats like .pub, .odt, .epub not supported
- Files that worked before suddenly stop opening
- Error messages with no helpful guidance

> *"I'm unable to open my most important doc so it doesn't work for me 🥺"*
> *"Bad experience — XLS sheets are not open"*

---

### 🔴 #5 — File Format Support Missing (4.2% — 525 reviews)
- Missing support for: .pub, .odt, .epub, .csv, .rtf, .psb
- RTL languages (Hebrew, Arabic) not rendered correctly
- Claims "all documents" but many formats fail

> *"Didn't meet my expectation.. can't open an epub file.."*

---

### 🔴 #6 — Fake / Misleading App (4.0% — 498 reviews)
- App described as "scam" or "waste of time"
- Features advertised but not delivered
- Users feel tricked into downloading

> *"This is worst app. It shows ads and diverts attention. Don't download this app."*
